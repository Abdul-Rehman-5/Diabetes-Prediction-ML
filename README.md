# 🩺 Diabetes Risk Prediction System

An end-to-end **Machine Learning project** that predicts diabetes risk using patient health indicators and Logistic Regression.

## 🔍 Project Overview
This project focuses on building a **real-world healthcare ML application**, covering the complete pipeline:
- Data preprocessing & cleaning
- Feature scaling
- Model training & evaluation
- Deployment as an interactive web app

- **Evaluation Metrics:** Recall, Confusion Matrix, Classification Report

## 🧠 Features Used
- Pregnancies
- Glucose Level
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

## 🖥️ Web Application
The trained model is deployed using **Streamlit**, allowing users to:
- Enter health parameters
- Get real-time diabetes risk prediction
- View probability-based risk assessment

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Streamlit
- Matplotlib, Seaborn
